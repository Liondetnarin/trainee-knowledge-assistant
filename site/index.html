Trainee Knowledge Assistant

เดชนรินทร์ ไชยอักษร

งานส่งทั้ง 4 Parts - เว็บแอป (Next.js + AI), วิดีโอ business scenario, แผน onboard trainee และแนวคิดต่อ legacy system - รวมอยู่ในหน้าเดียวเพื่อให้อ่านง่าย

Part 1 - Coding Task
Mini Knowledge Assistant: login, upload PDF/TXT, chat กับ AI ด้วย context จากเอกสาร, token counter - พร้อมวิธี install & setup ด้วย docker compose คำสั่งเดียว
Part 2 - Business Scenario Video
วิดีโอ 3 นาที รับมือลูกค้าที่โกรธตอน demo - ดูได้ในหน้าเว็บเลย พร้อมโครงเนื้อหาที่ใช้
Part 3 - Leadership Reflection
แผน onboard Trainee 5 คนตลอด 4 เดือน - week-by-week, KPI ที่วัดได้, วิธีจัดการ low performer และ Plan B
Part 4 - Legacy System Thinking
3 วันแรกกับระบบเก่า 10 ปีที่ไม่มีเอกสาร และการตัดสินใจเมื่อเจอไฟล์ .php แปลกใน server

Quick Start (Part 1)

git clone https://github.com/Liondetnarin/trainee-knowledge-assistant.git
cd trainee-knowledge-assistant
cp .env.example .env   # ใส่ AI_PROVIDER, AI_API_KEY, AI_MODEL
docker compose up

เปิด http://localhost:3000
